# PayPay to MoneyForward 仕分けツール

PayPayの取引明細CSVを取り込み、Webブラウザ上で仕分け（勘定科目の分類や事業・プライベートの選別）を行い、**マネーフォワード確定申告**でそのまま取り込める仕訳帳形式のCSVファイル（Shift-JIS / CRLF）としてエクスポートするツールです。

## 主な機能

- **📁 自動読み込み & アップロード**: `data/` フォルダ配下にあるPayPayのCSV取引履歴を自動検知して読み込むほか、ブラウザ上からの直接アップロードにも対応。
- **⚡ 自動仕分けルール**: 事前定義したキーワード（「ローソン」「Uber Eats」「Apple」など）に基づいて、勘定科目・税区分を自動判定。カスタムルールの追加・編集・削除もWeb UI上から行えます。
- **📝 インタラクティブ編集シート**: Streamlitの `st.data_editor` を使用し、スプレッドシート（Excel）ライクにブラウザ上で直接仕訳を編集可能。
- **💾 進捗の自動保存 (チェックポイント)**: 編集作業中のデータは自動的に `data/*_checkpoint.csv` に保存されるため、ブラウザを閉じても前回の続きから再開できます。
- **📈 グラフ & 分析レポート**: 申告対象の経費内訳（パイチャート）や月別の経費推移（スタックバーチャート）を Plotly でリアルタイムに可視化。
- **📤 MF形式エクスポート**: マネーフォワード確定申告が要求するShift-JISエンコーディングかつCRLF改行コードのCSV形式（27カラム構成）でエクスポート可能。

## 起動方法

[uv](https://github.com/astral-sh/uv) を使用して、仮想環境の同期から起動までワンコマンドで行うことができます。

### 1. 依存関係のセットアップ
```bash
uv sync
```

### 2. アプリケーションの起動
```bash
uv run streamlit run app.py
```

起動すると、ブラウザが自動的に開き `http://localhost:8501` でツールを使用できるようになります。

## 使い方フロー

1. **データの配置**: 
   PayPayアプリの取引履歴から出力したCSVファイルを `data/` ディレクトリ直下に置くか、画面上で直接ドラッグ＆ドロップしてアップロードします。
2. **自動ルール適用**:
   読み込まれたデータに対して、サイドバーの設定（チャージやポイント運用の除外）および `rules.json` のキーワードに基づいたルールが適用され、初期状態が作成されます。
3. **仕分けの確認と編集**:
   - 表の中で、各取引の「借方勘定科目」（経費科目）や「対象外」チェックボックスを調整します。
   - プライベートな取引は「対象外」にチェックを入れるだけで、最終出力から除外されます。
   - 編集が完了したら、表の下の **「変更を保存」** ボタンをクリックします。
4. **自動ルールの追加 (任意)**:
   - 「自動仕分けルール」タブから、繰り返し登場する店舗やサービスの仕分けルールを設定すると、次回から自動で分類されるようになります。
5. **CSVのダウンロード**:
   - 仕分け完了後、エディタタブ下部の **「マネーフォワード用仕訳CSVをダウンロード」** ボタンをクリックします。
   - 生成されたCSVは自動的にローカルの `output/` フォルダにも保存されます。

- [app.py](file:///Users/yoheiohto/github/paypay_with_moneyforword_kakuteishinkoku/app.py): アプリケーションのメインロジック（UI、ルール処理、エクスポートなど）

- `output/`: マネーフォワード取込用にエクスポートされたCSVファイルの保存先
